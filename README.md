# CS131Project

For team members: 
Make sure to have git working and setup on your computer/ide.
Install NodeJS. From: https://nodejs.org/en/download

To clone the repository and install required packages do:
git clone https://github.com/lanechavez19-hash/CS131Project.git
cd CS131Project          To go to the project folder
npm install              To install necessary packages i.e. Phaser + Vite 
npm run dev        This should create a localhost link for you to test the website.

If npm restricted write in Powershell :
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser


Before start coding for the day do:
git pull
For latest updates.

After finishing your changes for the day do:
git add .   (if you are adding more files to the project)
git commit - "Your message"
git push
