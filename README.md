# CS131Project

For team members: 
Make sure to have git working and setup on your computer/ide.
To clone the repository and install required packages do:
git clone https://github.com/lanechavez19-hash/CS131Project.git
cd CS131Project
npm install
npm run dev

Before start coding for the day do:
git pull
For latest updates.

After finishing your changes for the day do:
git add.
git commit - "Your message"
git push
